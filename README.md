# GearRent Pro 🎥🚀

**GearRent Pro** is a **smart, multi-branch equipment rental system** designed for companies that rent professional gear like cameras, lenses, drones, lighting kits, and audio equipment. It’s built to streamline operations, maximize efficiency, and provide an excellent experience for both staff and customers.  

## Key Features
- Manage multiple branches, each with its own equipment inventory.  
- Allow customers to **reserve and rent equipment** with real-time availability checks.  
- Track **returns, damages, late fees, and security deposits** automatically.  
- Offer **membership-based discounts** and promotions to loyal customers.  
- Generate insightful **reports** on revenue and equipment usage.  
- Ensure **role-based access**: Admins, Branch Managers, and Staff can perform their tasks securely.  

## Technology & Architecture
- **Frontend:** JavaFX or Java Swing  
- **Backend:** Java with JDBC  
- **Database:** MySQL  
- **Architecture:** Layered (Presentation → Controller → Service → DAO → Entity)  

Whether you’re managing one branch or multiple, GearRent Pro makes running your equipment rental business **fast, reliable, and stress-free**!
